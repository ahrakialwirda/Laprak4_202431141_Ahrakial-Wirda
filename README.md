# Laporan Praktikum 4

Program Penilaian Mahasiswa berbasis Web menggunakan PHP.

## Fitur
- Input nilai kehadiran, tugas, UTS, dan UAS
- Perhitungan nilai akhir
- Menentukan grade dan status kelulusan

## Cara Menjalankan
1. Simpan file di folder htdocs
2. Jalankan Apache di XAMPP
3. Akses melalui browser:
   http://localhost/nama_folder/polos.php
